\begin{designnote}
Modal captures are best-effort: each modal has a distinct trigger and some
open only when specific data is selected. If a modal is not captured, the
placeholder will read ``Screenshot pending''. The spec catalogues which
modals it attempts in \texttt{apps/frontend/e2e/capture/A-appendix-modals.spec.ts}.
\end{designnote}
